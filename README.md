# ADS Server

Run in Python3.8

### How to use it 

#### Run api in local environment

* At the root folder of the project

Windows 10
```
py -m Serveur.api
```

Linux system
```
python3.8 -m Serveur.api
```

#### Run sensor manager in local environment
* At the root folder of the project

Windows 10
```
py -m Serveur.sensor_manager
```

Linux system
```
python3.8 -m Serveur.sensor_manager
```
