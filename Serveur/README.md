# ADS Server

Run in Python3.8

### How to use it 

#### Run api in local environment

* At the root folder of the project

Windows 10
```
py -m api
```

Linux system
```
python3.8 -m api
```

#### Run sensor manager in local environment
* At the root folder of the project

Windows 10
```
py -m sensor_manager
```

Linux system
```
python3.8 -m sensor_manager
```
